# Microsoft.PowerShell.Core-Modul

## BESCHREIBUNG

Dieser Abschnitt enthält die Hilfe Themen für die Cmdlets, die mit dem PowerShell-Modul Microsoft. PowerShell. Core installiert werden. Das Core-Modul enthält Cmdlets und Anbieter, die die grundlegenden Features von PowerShell verwalten.

## Microsoft. PowerShell. Core-Cmdlets

### [Add-History](Add-History.md)
Fügt Einträge an den Sitzungsverlauf an.

### [Clear-History](Clear-History.md)
Löscht Einträge aus dem PowerShell-Sitzungs Befehlsverlauf.

### [Clear-Host](Clear-Host.md)
Löscht die Anzeige im Hostprogramm.

### [Connect-PSSession](Connect-PSSession.md)
Stellt die Verbindung mit getrennten Sitzungen wieder her.

### [Debug-Job](Debug-Job.md)
Debuggen eines laufenden Hintergrund-, Remote-oder PowerShell-Workflow Auftrags.

### [Disable-ExperimentalFeature](Disable-ExperimentalFeature.md)
Deaktivieren Sie eine experimentelle Funktion beim Starten einer neuen Instanz von PowerShell.

### [Disable-PSRemoting](Disable-PSRemoting.md)
Verhindert, dass PowerShell-Endpunkte Remote Verbindungen empfangen.

### [Disable-PSSessionConfiguration](Disable-PSSessionConfiguration.md)
Deaktiviert die Sitzungskonfigurationen auf dem lokalen Computer.

### [Disconnect-PSSession](Disconnect-PSSession.md)
Trennt eine Verbindung.

### [Enable-ExperimentalFeature](Enable-ExperimentalFeature.md)
Aktivieren Sie eine experimentelle Funktion beim Starten einer neuen Instanz von PowerShell.

### [Enable-PSRemoting](Enable-PSRemoting.md)
Konfiguriert den Computer für den Empfang von Remotebefehlen.

### [Enable-PSSessionConfiguration](Enable-PSSessionConfiguration.md)
Aktiviert die Sitzungskonfigurationen auf dem lokalen Computer.

### [Enter-PSHostProcess](Enter-PSHostProcess.md)
Stellt eine Verbindung mit einer interaktiven Sitzung mit einem lokalen Prozess her und wechselt in diese.

### [Enter-PSSession](Enter-PSSession.md)
Startet eine interaktive Sitzung mit einem Remotecomputer.

### [Exit-PSHostProcess](Exit-PSHostProcess.md)
Schließt eine interaktive Sitzung mit einem lokalen Prozess.

### [Exit-PSSession](Exit-PSSession.md)
Beendet eine interaktive Sitzung mit einem Remotecomputer.

### [Export-ModuleMember](Export-ModuleMember.md)
Gibt die Modulelemente an, die exportiert werden.

### [ForEach-Object](ForEach-Object.md)
Führt einen Vorgang für jedes Element in einer Auflistung von Eingabeobjekten aus.

### [Get-Command](Get-Command.md)
Ruft alle Befehle ab.

### [Get-ExperimentalFeature](Get-ExperimentalFeature.md)
Ruft experimentelle Funktionen ab.

### [Get-Help](Get-Help.md)
Zeigt Informationen zu PowerShell-Befehlen und-Konzepten an.

### [Get-History](Get-History.md)
Ruft eine Liste der Befehle ab, die während der aktuellen Sitzung eingegeben wurden.

### [Get-Job](Get-Job.md)
Ruft PowerShell-Hintergrund Aufträge ab, die in der aktuellen Sitzung ausgeführt werden.

### [Get-Module](Get-Module.md)
Ruft die Module ab, die in die aktuelle Sitzung importiert wurden oder importiert werden können.

### [Get-PSHostProcessInfo](Get-PSHostProcessInfo.md)
Ruft Prozessinformationen zum PowerShell-Host ab.

### [Get-PSSession](Get-PSSession.md)
Ruft die PowerShell-Sitzungen auf lokalen Computern und Remote Computern ab.

### [Get-PSSessionCapability](Get-PSSessionCapability.md)
Ruft die Funktionen eines bestimmten Benutzers in einer eingeschränkten Sitzungs Konfiguration ab.

### [Get-PSSessionConfiguration](Get-PSSessionConfiguration.md)
Ruft die registrierten Sitzungskonfigurationen auf dem Computer ab.

### [Import-Module](Import-Module.md)
Fügt der aktuellen Sitzung Module hinzu.

### [Invoke-Command](Invoke-Command.md)
Führt Befehle auf lokalen Computern und Remotecomputern aus.

### [Invoke-History](Invoke-History.md)
Führt Befehle aus dem Sitzungsverlauf aus.

### [New-Module](New-Module.md)
Erstellt ein neues dynamisches Modul, das nur im Arbeitsspeicher vorhanden ist.

### [New-ModuleManifest](New-ModuleManifest.md)
Erstellt ein neues Modulmanifest.

### [New-PSRoleCapabilityFile](New-PSRoleCapabilityFile.md)
Erstellt eine Datei, die einen Satz von Funktionen definiert, die durch eine Sitzungs Konfiguration verfügbar gemacht werden sollen.

### [New-PSSession](New-PSSession.md)
Erstellt eine permanente Verbindung mit einem lokalen Computer oder Remotecomputer.

### [New-PSSessionConfigurationFile](New-PSSessionConfigurationFile.md)
Erstellt eine Datei, die eine Sitzungskonfiguration definiert.

### [New-PSSessionOption](New-PSSessionOption.md)
Erstellt ein Objekt, das erweiterte Optionen für eine PSSession enthält.

### [New-PSTransportOption](New-PSTransportOption.md)
Erstellt ein Objekt, das erweiterte Optionen für eine Sitzungskonfiguration enthält.

### [Out-Default](Out-Default.md)
Sendet die Ausgabe an das Standardformatierungsprogramm und das Standard-Ausgabe-Cmdlet.

### [Out-Host](Out-Host.md)
Sendet eine Ausgabe an die Befehlszeile.

### [Out-Null](Out-Null.md)
Blendet die Ausgabe aus, anstatt Sie über die Pipeline zu senden oder anzuzeigen.

### [Receive-Job](Receive-Job.md)
Ruft die Ergebnisse der PowerShell-Hintergrund Aufträge in der aktuellen Sitzung ab.

### [Receive-PSSession](Receive-PSSession.md)
Ruft die Ergebnisse von Befehlen in getrennten Sitzungen ab.

### [Register-ArgumentCompleter](Register-ArgumentCompleter.md)
Registriert ein benutzerdefiniertes Argument, das vervollständigt werden soll.

### [Register-PSSessionConfiguration](Register-PSSessionConfiguration.md)
Erstellt und registriert eine neue Sitzungskonfiguration.

### [Remove-Job](Remove-Job.md)
Löscht einen PowerShell-Hintergrund Auftrag.

### [Remove-Module](Remove-Module.md)
Entfernt Module aus der aktuellen Sitzung.

### [Remove-PSSession](Remove-PSSession.md)
Schließt mindestens eine PowerShell-Sitzung (pssessions).

### [Save-Help](Save-Help.md)
Lädt die neuesten Hilfedateien herunter und speichert sie in einem Dateisystemverzeichnis.

### [Set-PSDebug](Set-PSDebug.md)
Aktiviert bzw. deaktiviert Skriptdebuggingfeatures, legt die Ablaufverfolgungsebene fest und schaltet den Strict-Modus um.

### [Set-PSSessionConfiguration](Set-PSSessionConfiguration.md)
Ändert die Eigenschaften einer registrierten Sitzungskonfiguration.

### [Set-StrictMode](Set-StrictMode.md)
Erstellt und erzwingt Codierungsregeln in Ausdrücken, Skripts und Skriptblöcken.

### [Start-Job](Start-Job.md)
Startet einen PowerShell-Hintergrund Auftrag.

### [Stop-Job](Stop-Job.md)
Beendet einen PowerShell-Hintergrund Auftrag.

### [Test-ModuleManifest](Test-ModuleManifest.md)
Überprüft, ob eine Modulmanifestdatei den Inhalt eines Moduls genau beschreibt.

### [Test-PSSessionConfigurationFile](Test-PSSessionConfigurationFile.md)
Überprüft die Schlüssel und Werte in einer Sitzungskonfigurationsdatei.

### [Unregister-PSSessionConfiguration](Unregister-PSSessionConfiguration.md)
Löscht registrierte Sitzungskonfigurationen vom Computer.

### [Update-Help](Update-Help.md)
Lädt die neuesten Hilfedateien auf den Computer herunter und installiert sie.

### [Wait-Job](Wait-Job.md)
Unterdrückt die Eingabeaufforderung, bis ein oder alle PowerShell-Hintergrund Aufträge, die in der Sitzung ausgeführt werden, abgeschlossen sind.

### [Where-Object](Where-Object.md)
Wählt Objekte aus einer Auflistung basierend auf ihren Eigenschaftswerten aus.
