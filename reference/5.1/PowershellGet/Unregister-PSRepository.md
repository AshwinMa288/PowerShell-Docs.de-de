# Unregister-PSRepository

## ZUSAMMENFASSUNG
Hebt die Registrierung eines Repositorys auf

## SYNTAX

```
Unregister-PSRepository [-Name] <String[]> [<CommonParameters>]
```

## DESCRIPTION
