# <a name="format-schema-xml-reference"></a>Formatschema – XML-Referenz

In den Themen in diesem Abschnitt werden die XML-Elemente beschrieben, die beim Formatieren von Dateien (Format. ps1xml files) verwendet werden. Formatieren von Dateien definieren Sie, wie das .NET-Objekt angezeigt wird. das Objekt selbst wird nicht geändert.
