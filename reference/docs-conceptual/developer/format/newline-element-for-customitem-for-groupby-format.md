# <a name="newline-element-for-customitem-for-groupby-format"></a>Element „NewLine“ für CustomItem für GroupBy (Format)

Fügt der Anzeige des-Steuer Elements eine leere Zeile hinzu. Dieses Element wird verwendet, wenn definiert wird, wie eine neue Gruppe von Objekten angezeigt wird.

Konfigurationselement (Format) viewdefinitions-Element (Format) Ansichts Element (Format) GroupBy-Element für View (Format) CustomControl-Element für GroupBy (Format) customentries-Element für CustomControl für GroupBy (Format) customentry-Element für CustomControl für GroupBy (Format) customItem-Element für customentry für GroupBy (Format) newline-Element für customItem für GroupBy (Format)

## <a name="syntax"></a>Syntax

```xml
<NewLine/>
```
