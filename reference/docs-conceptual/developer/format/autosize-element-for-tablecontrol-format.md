# <a name="autosize-element-for-tablecontrol-format"></a>Element „AutoSize“ für TableControl (Format)

Gibt an, ob die Spaltengröße und die Anzahl der Spalten basierend auf der Größe der Daten angepasst werden.

Viewdefinitions-Element (Format) View-Element (Format) tablecontrol-Element (Format) AutoSize-Element für tablecontrol (Format)

## <a name="syntax"></a>Syntax

```xml
<AutoSize/>
```
