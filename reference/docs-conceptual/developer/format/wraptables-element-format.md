# <a name="wraptables-element-format"></a>Element „WrapTables“ (Format)

Gibt an, dass Daten in einer Tabellenzelle in die nächste Zeile verschoben werden, wenn die Daten länger als die Spaltenbreite sind.

Configuration-Element (Format) DefaultSettings-Element (Format) wraptables-Element (Format)

## <a name="syntax"></a>Syntax

```xml
<WrapTables/>
```

## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete Elemente und das übergeordnete Element des- `WrapTables` Elements beschrieben.

### <a name="attributes"></a>Attribute

Keine

### <a name="child-elements"></a>Untergeordnete Elemente

Keine
