# <a name="text-element-for-customitem-for-groupby-format"></a>Element „Text“ für CustomItem für GroupBy (Format)

Gibt Text an, der den vom Steuerelement angezeigten Daten hinzugefügt wird, z. b. eine Bezeichnung, eckige Klammern zum Einschließen der Daten und Leerzeichen, um die Daten einzuschließen. Dieses Element wird verwendet, wenn definiert wird, wie eine neue Gruppe von Objekten angezeigt wird.

Konfigurationselement (Format) viewdefinitions-Element (Format) Ansichts Element (Format) GroupBy-Element für View (Format) CustomControl-Element für GroupBy (Format) customentries-Element für CustomControl für GroupBy (Format) customentry-Element für CustomControl für GroupBy (Format) customItem-Element für customentry für GroupBy (Format) Text Element für customItem für GroupBy (Format)

## <a name="syntax"></a>Syntax

```xml
<Text>TextToDisplay</Text>
```

## <a name="attributes-and-elements"></a>Attribute und Elemente

In den folgenden Abschnitten werden Attribute, untergeordnete Elemente und das übergeordnete Element des `Text`-Elements beschrieben.

### <a name="attributes"></a>Attributes

Keine.

### <a name="child-elements"></a>Untergeordnete Elemente

Keine.

### <a name="parent-elements"></a>Übergeordnete Elemente

|Element|Description|
|-------------|-----------------|
|[CustomItem-Element für customentry für GroupBy (Format)](./customitem-element-for-customentry-for-groupby-format.md)|Definiert ein Steuerelement für die benutzerdefinierte Steuerelement Ansicht.|

## <a name="text-value"></a>Textwert

Geben Sie den Text eines Steuer Elements für die anzuzeigenden Daten an.
