# <a name="propertyname-element-for-expressionbinding-for-customcontrol-for-view-format"></a>Element „PropertyName“ für ExpressionBinding für CustomControl für View (Format)

Gibt die .net-Eigenschaft an, deren Wert vom-Steuerelement angezeigt wird. Dieses Element wird beim Definieren einer benutzerdefinierten Steuerelement Ansicht verwendet.

Konfigurationselement (Format) viewdefinitions-Element (Format) View-Element (Format) CustomControl-Element für View (Format) customentries-Element für CustomControl for View (Format) customentry-Element für customentries für CustomControl for View (Format) customItem-Element für customentry für CustomControl für das View (Format) ExpressionBinding-Element für customItem für CustomControl für View (Format) propertyName-Element für ExpressionBinding für CustomControl for View (Format)

## <a name="syntax"></a>Syntax

```xml
<PropertyName>.NetTypeProperty</PropertyName>
```

Keine

### <a name="child-elements"></a>Untergeordnete Elemente

Keine

## <a name="text-value"></a>Textwert

Geben Sie den Namen der .net-Eigenschaft an, deren Wert vom Steuerelement angezeigt wird.
