# <a name="enumeratecollection-element-for-expressionbinding-for-controls-for-view-format"></a>Element „EnumerateCollection“ für ExpressionBinding für Controls für View (Format)

Gibt an, dass die Elemente der Auflistungen angezeigt werden. Dieses Element wird beim Definieren von Steuerelementen verwendet, die von einer Ansicht verwendet werden können.
