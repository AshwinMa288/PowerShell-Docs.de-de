# <a name="typename-element-for-selectioncondition-for-controls-for-view-format"></a>Element „TypeName“ für SelectionCondition für Controls für View (Format)

Gibt einen .NET-Typ an, der die Bedingung auslöst. Dieses Element wird beim Definieren von Steuerelementen verwendet, die von einer Ansicht verwendet werden können.
