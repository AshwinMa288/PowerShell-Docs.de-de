# <a name="scriptblock-element-for-expressionbinding-for-controls-for-view-format"></a>Element „ScriptBlock“ für ExpressionBinding für Controls für View (Format)

Gibt das Skript an, dessen Wert vom-Steuerelement angezeigt wird. Dieses Element wird beim Definieren von Steuerelementen verwendet, die von einer Ansicht verwendet werden können.
