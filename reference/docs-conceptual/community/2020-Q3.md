# <a name="whats-new-in-powershell-docs"></a>Neuerungen in der PowerShell-Dokumentation

In diesem Artikel sind einige der wichtigsten Änderungen an den Dokumenten im vergangenen Monat aufgeführt. Außerdem werden die Beiträge der Community gewürdigt.

Helfen Sie uns, die Dokumentation zu verbessern. Lesen Sie den [Leitfaden für Mitwirkende], um mehr über die ersten Schritte zu erfahren.

## <a name="2020-august-highlights"></a>Highlights im August 2020

- Neue PowerShell-Dokumentation
  - [About_Calculated_Properties](/powershell/module/microsoft.powershell.core/about/about_calculated_properties)
  - [Verwenden von „Write-Progress“ über mehrere Threads mit dem Parameter „Parallel“ im Cmdlet „Foreach-Object“](/powershell/scripting/learn/deep-dives/write-progress-across-multiple-threads)
- Korrekturen in Dokumentationen
  - Aktualisierungen für Inhalte zur 7.1-Vorschauversion

- GitHub-Statistik
  - 69 Pull Requests zusammengeführt (26 aus der Community)
  - 68 Issues geöffnet (49 aus der Community)
  - 58 Issues geschlossen

## <a name="2020-july-highlights"></a>Highlights im Juli 2020

- Neue PowerShell-Dokumentation
  - Hervorgeholte alte [ETS-Dokumente](/powershell/scripting/developer/ets/overview):7 Artikel hinzugefügt
  - Artikel zum [Erstellen aktualisierbarer Hilfe mithilfe von PlatyPS](/powershell/scripting/dev-cross-plat/create-help-using-platyps) hinzugefügt
- Korrekturen in Dokumentationen
  - Aktualisierungen für Inhalte zur 7.1-Vorschauversion
  - Seitenkopf aktualisiert: Menüoptionen vereinfacht und Schaltfläche „Herunterladen“ hinzugefügt
  - Mehrere Probleme mit `Update-Help` behoben
    - Hilfe zu den Modulen PSDesiredStateConfiguration und ThreadJob jetzt herunterladbar
    - Aktualisierbare Hilfe für PowerShell 7.1 Preview veröffentlicht
    - Aktualisierbare Hilfe für PowerShell 5.1 enthält jetzt Themen des Typs „Info“

- GitHub-Statistik
  - 99 Pull Requests zusammengeführt (29 aus der Community)
  - 51 Issues geöffnet (44 aus der Community)
  - 71 Issues geschlossen
