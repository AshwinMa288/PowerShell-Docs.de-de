# <a name="whats-new-in-powershell-docs"></a>Neuerungen in der PowerShell-Dokumentation

In diesem Artikel sind einige der wichtigsten Änderungen an den Dokumenten im vergangenen Monat aufgeführt. Außerdem werden die Beiträge der Community gewürdigt.

Helfen Sie uns, die Dokumentation zu verbessern. Lesen Sie den [Leitfaden für Mitwirkende][contrib], um mehr über die ersten Schritte zu erfahren.

## <a name="2020-june-highlights"></a>Highlights im Juni 2020

- Neue PowerShell-Dokumentation
  - Neuer Inhalt zu [PowerShell 101](../learn/ps101/00-introduction.md) von Mike F. Robbins veröffentlicht
  - Zwei aktuelle Blogbeiträge von Rob Holt zur Dokumentation von **Skripterstellung und Entwicklung** hinzugefügt
    - [Auswählen des richtigen PowerShell NuGet-Pakets für Ihr .NET-Projekt](/powershell/scripting/dev-cross-plat/choosing-the-right-nuget-package)
    - [Auflösen von Abhängigkeitskonflikten bei der Modulassembly in PowerShell](/powershell/scripting/dev-cross-plat/resolving-dependency-conflicts)
- Korrekturen in Dokumentationen
  - Archivierung älterer Inhalte in [https://aka.ms/PSLegacyDocs](https://aka.ms/PSLegacyDocs)
    - Inhalte zum PowerShell Web Access SDK
    - Inhalte zum PowerShell-Workflows SDK
  - Aktualisierungen für Inhalte zur 7.1-Vorschauversion

- GitHub-Statistik
  - 83 Pull Requests zusammengeführt (15 aus der Community)
  - 68 Issues geöffnet (52 aus der Community)
  - 74 Issues geschlossen
