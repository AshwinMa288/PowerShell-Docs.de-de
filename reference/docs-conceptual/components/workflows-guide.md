# <a name="psworkflow-module"></a>PSWorkflow-Modul

Dieser Abschnitt enthält die Hilfethemen für das **PSWorkflow**-Modul, das Cmdlets enthält, die die Windows PowerShell-Workflowfunktion unterstützen. Das **PSWorkflow**-Modul wird mit Windows PowerShell 5.1 installiert.

> [!NOTE]
> Workflows werden nur in Windows PowerShell 5.1 unterstützt.
