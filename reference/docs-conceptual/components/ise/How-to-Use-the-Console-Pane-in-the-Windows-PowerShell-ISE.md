<span data-ttu-id="03167-104">Der Konsolenbereich in Windows PowerShell Integrated Scripting Environment (ISE) funktioniert genauso wie das eigenständige Windows PowerShell ISE-Konsolenfenster.</span><span class="sxs-lookup"><span data-stu-id="03167-104">The Console pane in the Windows PowerShell Integrated Scripting Environment (ISE) operates exactly like the stand-alone Windows PowerShell ISE console window.</span></span>

<span data-ttu-id="03167-105">Um einen Befehl im Konsolenbereich auszuführen, geben Sie einen Befehl ein und drücken dann die <kbd>EINGABETASTE</kbd>.</span><span class="sxs-lookup"><span data-stu-id="03167-105">To run a command in the Console Pane, type a command, and then press <kbd>ENTER</kbd>.</span></span> <span data-ttu-id="03167-106">Um mehrere Befehle einzugeben, die nacheinander ausgeführt werden sollen, drücken Sie <kbd>UMSCHALT</kbd>+<kbd>EINGABETASTE</kbd> zwischen den Befehlen.</span><span class="sxs-lookup"><span data-stu-id="03167-106">To enter multiple commands that you want to execute in sequence, type <kbd>SHIFT</kbd>+<kbd>ENTER</kbd> between commands.</span></span> <span data-ttu-id="03167-107">Hilfe zum Eingeben von Befehlen finden Sie unter [Verwenden von Vervollständigung mit der TAB-TASTE im Skriptbereich und Konsolenbereich](How-to-Use-Tab-Completion-in-the-Script-Pane-and-Console-Pane.md).</span><span class="sxs-lookup"><span data-stu-id="03167-107">See [How to Use Tab Completion in the Script Pane and Console Pane](How-to-Use-Tab-Completion-in-the-Script-Pane-and-Console-Pane.md) for help in typing commands.</span></span>

<span data-ttu-id="03167-108">Um einen Befehl zu beenden, klicken Sie auf der Symbolleiste auf **Vorgang beenden**, oder drücken Sie <kbd>STRG</kbd>+<kbd>UNTBR</kbd>.</span><span class="sxs-lookup"><span data-stu-id="03167-108">To stop a command, on the toolbar, click **Stop Operation**, or press <kbd>CTRL</kbd>+<kbd>BREAK</kbd>.</span></span> <span data-ttu-id="03167-109">Sie können auch <kbd>STRG</kbd>+<kbd>C</kbd> verwenden, um einen Befehl zu beenden, sofern der Kontext eindeutig ist.</span><span class="sxs-lookup"><span data-stu-id="03167-109">You can also use <kbd>CTRL</kbd>+<kbd>C</kbd> to stop a command if the context is unambiguous.</span></span> <span data-ttu-id="03167-110">Wenn im aktuellen Bereich beispielsweise Text ausgewählt ist,wird <kbd>STRG</kbd>+<kbd>C</kbd> dem Kopiervorgang zugeordnet.</span><span class="sxs-lookup"><span data-stu-id="03167-110">For example, if some text has been selected in the current Pane, then <kbd>CTRL</kbd>+<kbd>C</kbd> maps to the copy operation.</span></span>

<span data-ttu-id="03167-111">Seit Windows PowerShell Version 3 ist der Ausgabebereich mit dem Konsolenbereich kombiniert.</span><span class="sxs-lookup"><span data-stu-id="03167-111">Beginning in Windows PowerShell v3, the Output pane was combined with the Console pane.</span></span> <span data-ttu-id="03167-112">Dies hat den Vorteil, dass sich das System wie die eigenständige Windows PowerShell-Konsole verhält, und beseitigt die Unterschiede in den Vorgehensweisen, die erforderlich waren, als die Bereiche getrennt waren.</span><span class="sxs-lookup"><span data-stu-id="03167-112">This has the benefit of behaving like the stand-alone Windows PowerShell console and eliminates the differences in procedures that were needed when they were separate.</span></span> <span data-ttu-id="03167-113">Ihre Möglichkeiten:</span><span class="sxs-lookup"><span data-stu-id="03167-113">You can:</span></span>

- <span data-ttu-id="03167-118">Einfügen des markierten Texts an der aktuellen Cursorposition.</span><span class="sxs-lookup"><span data-stu-id="03167-118">Paste the selected text at a current cursor position.</span></span> <span data-ttu-id="03167-119">Klicken Sie auf der Symbolleiste auf das Symbol **Einfügen**.</span><span class="sxs-lookup"><span data-stu-id="03167-119">Click the **Paste** icon on the toolbar.</span></span>

- <span data-ttu-id="03167-120">Löschen des gesamten Texts im Konsolenbereich.</span><span class="sxs-lookup"><span data-stu-id="03167-120">Clear all the text in the Console pane.</span></span> <span data-ttu-id="03167-121">Sie können auf der Symbolleiste auf das Symbol **Konsolenbereich löschen** klicken, oder Sie führen den Befehl `Clear-Host` oder dessen Alias `cls` aus, um den Konsolenbereich zu löschen.</span><span class="sxs-lookup"><span data-stu-id="03167-121">To clear the Console pane, you can click the **Clear Console Pane** icon on the toolbar, or run the command `Clear-Host` or its alias, `cls`.</span></span>
