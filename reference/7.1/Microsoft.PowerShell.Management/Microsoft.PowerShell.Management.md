# Microsoft.PowerShell.Management-Modul

## BESCHREIBUNG

Dieser Abschnitt enthält die Hilfe Themen für die Cmdlets, die mit dem PowerShell-Modul Microsoft. PowerShell. Management installiert werden. Das Verwaltungsmodul enthält Cmdlets, die Sie bei der Verwaltung von Windows in PowerShell unterstützen.

## Microsoft. PowerShell. Management-Cmdlets

### [Add-Content](Add-Content.md)
Fügt den angegebenen Elementen Inhalt hinzu, z. B. Hinzufügen von Wörtern in einer Datei.

### [Clear-Content](Clear-Content.md)
Löscht den Inhalt eines Elements, jedoch nicht das Element selbst.

### [Clear-Item](Clear-Item.md)
Löscht den Inhalt eines Elements, löscht das Element jedoch nicht.

### [Clear-ItemProperty](Clear-ItemProperty.md)
Löscht den Wert einer Eigenschaft, löscht jedoch nicht die-Eigenschaft.

### [Clear-RecycleBin](Clear-RecycleBin.md)
Löscht den Inhalt eines Papierkorbs.

### [Convert-Path](Convert-Path.md)
Konvertiert einen Pfad von einem PowerShell-Pfad in einen PowerShell-Anbieter Pfad.

### [Copy-Item](Copy-Item.md)
Kopiert ein Element von einem Speicherort an einen anderen Speicherort.

### [Copy-ItemProperty](Copy-ItemProperty.md)
Kopiert eine Eigenschaft und einen Wert von einem bestimmten Speicherort an einen anderen Speicherort.

### [Debug-Process](Debug-Process.md)
Debuggt Prozesse, die auf dem lokalen Computer ausgeführt werden.

### [Get-ChildItem](Get-ChildItem.md)
Ruft die Elemente und untergeordneten Elemente an angegebenen Speicherorten ab.

### [Get-Clipboard](Get-Clipboard.md)
Ruft den Inhalt der Zwischenablage ab.  [!NOTE] > unter Linux erfordert dieses Cmdlet, dass sich das `xclip` Hilfsprogramm im Pfad befinden muss.

### [Get-ComputerInfo](Get-ComputerInfo.md)
Ruft ein konsolidiertes Objekt der System-und Betriebssystem Eigenschaften ab.

### [Get-Content](Get-Content.md)
Ruft den Inhalt des Elements am angegebenen Speicherort ab.

### [Get-HotFix](Get-HotFix.md)
Ruft die Hotfixes ab, die auf lokalen Computern oder Remote Computern installiert sind.

### [Get-Item](Get-Item.md)
Ruft das Element am angegebenen Speicherort ab.

### [Get-ItemProperty](Get-ItemProperty.md)
Ruft die Eigenschaften eines angegebenen Elements ab.

### [Get-ItemPropertyValue](Get-ItemPropertyValue.md)
Ruft den Wert für eine oder mehrere Eigenschaften eines angegebenen Elements ab.

### [Get-Location](Get-Location.md)
Ruft Informationen zum aktuellen Arbeitsspeicherot oder Speicherstapel ab.

### [Get-Process](Get-Process.md)
Ruft die Prozesse ab, die auf dem lokalen Computer ausgeführt werden.

### [Get-PSDrive](Get-PSDrive.md)
Ruft Laufwerke in der aktuellen Sitzung ab.

### [Get-PSProvider](Get-PSProvider.md)
Ruft Informationen zum angegebenen PowerShell-Anbieter ab.

### [Get-Service](Get-Service.md)
Ruft die Dienste auf dem Computer ab.

### [Get-TimeZone](Get-TimeZone.md)
Ruft die aktuelle Zeitzone oder eine Liste verfügbarer Zeitzonen ab.

### [Invoke-Item](Invoke-Item.md)
Führt die Standardaktion für das angegebene Element aus.

### [Join-Path](Join-Path.md)
Kombiniert einen Pfad und einen untergeordneten Pfad zu einem Pfad.

### [Move-Item](Move-Item.md)
Verschiebt ein Element von einem Speicherort an einen anderen Speicherort.

### [Move-ItemProperty](Move-ItemProperty.md)
Verschiebt eine Eigenschaft von einem Speicherort an einen anderen Speicherort.

### [New-Item](New-Item.md)
Erstellt ein neues Element.

### [New-ItemProperty](New-ItemProperty.md)
Erstellt eine neue Eigenschaft für ein Element und legt den Wert fest.

### [New-PSDrive](New-PSDrive.md)
Erstellt temporäre und permanente zugeordnete Netzwerklaufwerke.

### [New-Service](New-Service.md)
Erstellt einen neuen Windows-Dienst.

### [Pop-Location](Pop-Location.md)
Ändert den aktuellen Speicherort in den Speicherort, der zuletzt auf den Stapel verschoben wurde.

### [Push-Location](Push-Location.md)
Fügt den aktuellen Speicherort an oberster Stelle eines Speicherstapels hinzu.

### [Remove-Item](Remove-Item.md)
Löscht die angegebenen Elemente.

### [Remove-ItemProperty](Remove-ItemProperty.md)
Löscht die Eigenschaft und deren Wert aus einem Element.

### [Remove-PSDrive](Remove-PSDrive.md)
Löscht temporäre PowerShell-Laufwerke und trennt die Verbindung mit zugeordneten Netzlaufwerken.

### [Remove-Service](Remove-Service.md)
Entfernt einen Windows-Dienst.

### [Rename-Computer](Rename-Computer.md)
Benennt einen Computer um.

### [Rename-Item](Rename-Item.md)
Benennt ein Element in einem PowerShell-Anbieter Namespace um.

### [Rename-ItemProperty](Rename-ItemProperty.md)
Benennt die Eigenschaft eines Elements um.

### [Resolve-Path](Resolve-Path.md)
Löst die Platzhalterzeichen in einem Pfad auf und zeigt den Inhalt des Pfads an.

### [Restart-Computer](Restart-Computer.md)
Startet das Betriebssystem auf lokalen Computern und Remote Computern neu.

### [Restart-Service](Restart-Service.md)
Beendet Dienste und startet diese anschließend.

### [Resume-Service](Resume-Service.md)
Setzt angehaltene (unterbrochene) Dienste fort.

### [Set-Clipboard](Set-Clipboard.md)
Legt den Inhalt der Zwischenablage fest.

### [Set-Content](Set-Content.md)
Schreibt neuen Inhalt oder ersetzt vorhandene Inhalte in einer Datei.

### [Set-Item](Set-Item.md)
Ändert den Wert eines Elements in den Wert, der im Befehl angegeben wird.

### [Set-ItemProperty](Set-ItemProperty.md)
Erstellt oder ändert den Wert für eine Eigenschaft eines Elements.

### [Set-Location](Set-Location.md)
Legt den aktuellen Arbeitsspeicherort auf einen angegebenen Speicherort fest.

### [Set-Service](Set-Service.md)
Startet, beendet und hält einen Dienst an und ändert seine Eigenschaften.

### [Set-TimeZone](Set-TimeZone.md)
Legt die Zeitzone des Systems auf eine angegebene Zeitzone fest.

### [Split-Path](Split-Path.md)
Gibt den angegebenen Teil eines Pfads zurück.

### [Start-Process](Start-Process.md)
Startet Prozesse auf dem lokalen Computer.

### [Start-Service](Start-Service.md)
Startet beendete Dienste.

### [Stop-Computer](Stop-Computer.md)
Beendet den lokalen Computer und Remotecomputer (fährt sie herunter).

### [Stop-Process](Stop-Process.md)
Beendet ausgeführte Prozesse.

### [Stop-Service](Stop-Service.md)
Beendet ausgeführte Dienste.

### [Suspend-Service](Suspend-Service.md)
Hält ausgeführte Dienste an.

### [Test-Connection](Test-Connection.md)
Sendet ICMP-Echo Anforderungs Pakete (Pings) an einen oder mehrere Computer.

### [Test-Path](Test-Path.md)
Bestimmt, ob alle Elemente eines Pfads vorhanden sind.

### [Wait-Process](Wait-Process.md)
Wartet, bis die Prozesse beendet wurden, bevor weitere Eingaben angenommen werden.
