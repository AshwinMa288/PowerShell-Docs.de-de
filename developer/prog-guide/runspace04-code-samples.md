# <a name="runspace04-code-samples"></a>Runspace04-Codebeispiele

Hier ist ein Codebeispiel für einen Runspace, die verwendet die [System.Management.Automation.Runspaceinvoke](/dotnet/api/System.Management.Automation.RunspaceInvoke) Klasse, um ein Skript ausführen, das einen Abbruchfehler generiert. Die hostanwendung ist verantwortlich für den Fehler abfangen und Interpretieren der Error-Datensatzes.

> [!NOTE]
> Sie können die VB.NET-Quelldatei (Runspace04.vb) für diesen Runspace mithilfe des Windows Software Development Kit für Windows Vista und Microsoft .NET Framework 3.0-Laufzeitkomponenten herunterladen. Anweisungen zum Herunterladen, finden Sie unter [das Installieren von Windows PowerShell und das Windows PowerShell-SDK-Download](/powershell/developer/installing-the-windows-powershell-sdk).
