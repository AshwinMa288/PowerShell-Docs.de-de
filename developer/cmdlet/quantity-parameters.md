# <a name="quantity-parameters"></a>Mengenparameter

Die folgende Tabelle enthält die empfohlenen Namen und die Funktionen für Mengenparameter.

|Parameter|Funktion|
|---|---|
|**Alle**<br>Datentyp: Boolescher Wert|Implementieren Sie diesen Parameter, damit `true` gibt an, dass alle Ressourcen anstelle einer Standardteilmenge von Ressourcen liegende sollte. Implementieren Sie diesen Parameter, damit `false` gibt eine Teilmenge der Ressourcen.|
|**Allocation**<br>Datentyp: Int32|Implementieren Sie diesen Parameter, damit der Benutzer auf die Anzahl von Elementen zuweisen angeben kann.|
|**BlockCount**<br>Datentyp: Int64|Implementieren Sie diesen Parameter, damit der Benutzer auf die Anzahl der Block angeben kann.|
|**Count**<br>Datentyp: Int64|Implementieren Sie diesen Parameter, damit der Benutzer auf die Anzahl die angeben kann.|
|**Bereich**<br>Datentyp: Schlüsselwort|Implementieren Sie diesen Parameter, damit der Benutzer den Bereich der zu verarbeitende angeben kann.|
