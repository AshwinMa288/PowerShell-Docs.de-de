# <a name="scriptblock-element-for-expressionbinding-for-controls-for-configuration-format"></a>Element „ScriptBlock“ für ExpressionBinding für Controls für Configuration (Format)

Gibt das Skript an, dessen Wert durch das allgemeine Steuerelement angezeigt wird. Dieses Element wird verwendet, wenn ein allgemeines Steuerelement zu definieren, das von allen Ansichten im die Formatierungsdatei verwendet werden kann.

## <a name="syntax"></a>Syntax

```xml
<ScriptBlock>ScriptToEvaluate</ScriptBlock>
```
