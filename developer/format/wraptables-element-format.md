# <a name="wraptables-element-format"></a>Element „WrapTables“ (Format)

Gibt an, dass die Daten in eine Tabellenzelle in der nächsten Zeile verschoben werden, wenn die Daten länger als die Breite der Spalte ist.

-Element (Format) DefaultSettings-Element (Format) WrapTables Konfigurationselement (Format)

## <a name="syntax"></a>Syntax

```xml
<WrapTables/>
```

## <a name="attributes-and-elements"></a>Attribute und Elemente

Den folgenden Abschnitten werden Attribute, untergeordnete Elemente und das übergeordnete Element des der `WrapTables` Element.

### <a name="attributes"></a>Attributes

Keine.

### <a name="child-elements"></a>Untergeordnete Elemente

Keine.

### <a name="parent-elements"></a>Übergeordnete Elemente

|Element|Beschreibung|
|-------------|-----------------|
|[DefaultSettings-Element (Format)](./defaultsettings-element-format.md)|Definiert allgemeine Einstellungen, die für alle Ansichten die Formatierungsdatei gelten.|
