# <a name="scriptblock-element-for-tablecolumnitem-for-tablecontrol-format"></a>Element „ScriptBlock“ für TableColumnItem für TableControl (Format)

Gibt das Skript an, dessen Wert in der Spalte der Zeile angezeigt wird.
