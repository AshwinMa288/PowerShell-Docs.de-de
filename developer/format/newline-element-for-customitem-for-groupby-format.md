# <a name="newline-element-for-customitem-for-groupby-format"></a>Element „NewLine“ für CustomItem für GroupBy (Format)

Eine leere Zeile eingefügt, um die Anzeige des Steuerelements. Dieses Element wird verwendet, wenn Sie definieren, wie eine neue Gruppe von Objekten angezeigt wird.

-Element (Format) ViewDefinitions-Element (Format) anzeigen (Format)-Element GroupBy Konfigurationselement für (Format) CustomControl Ansichtselement für GroupBy (Format) CustomEntries-Element für CustomControl für GroupBy (Format) CustomEntry-Element für CustomControl für GroupBy (Format) CustomItem-Element für CustomEntry für GroupBy (Format) neue-Zeile-Element für CustomItem für GroupBy (Format)

## <a name="syntax"></a>Syntax

```xml
<NewLine/>
```
