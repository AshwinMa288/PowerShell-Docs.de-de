Gibt den Text, der hinzugefügt wird, auf die Daten, die durch das Steuerelement wie einer Bezeichnung angezeigt wird, Klammern, um die Daten und Leerzeichen für den Einzug der das zu schließen. Dieses Element wird verwendet, wenn Sie definieren, wie eine neue Gruppe von Objekten angezeigt wird.
