# <a name="newline-element-for-customitem-for-controls-for-configuration-format"></a>Element „NewLine“ für CustomItem für Controls für Configuration (Format)

Eine leere Zeile eingefügt, um die Anzeige des Steuerelements. Dieses Element wird verwendet, wenn ein allgemeines Steuerelement zu definieren, das von allen Ansichten im die Formatierungsdatei verwendet werden kann.

Konfigurationselement (Format)-Element-Steuerelemente (Format)-Steuerelement Konfigurationselements für Steuerelemente für (Format) CustomControl Konfigurationselement für Steuerelement (Format) CustomEntries Konfigurationselement für CustomControl für Konfiguration ( CustomEntry-Element für CustomControl für Steuerelemente für (Format) CustomItem Konfigurationselement für CustomEntry für Steuerelemente für neue-Zeile-Konfigurationselement für CustomItem für Steuerelemente für die Konfiguration (Format)-Format)

## <a name="syntax"></a>Syntax

```xml
<NewLine/>
```
