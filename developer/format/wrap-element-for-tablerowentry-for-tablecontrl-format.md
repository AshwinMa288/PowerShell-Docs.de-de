# <a name="wrap-element-for-tablerowentry-for-tablecontrl--format"></a>Element „Wrap“ für TableRowEntry für TableControl (Format)

Gibt an, dass Text, der die Spaltenbreite überschreitet, die in der nächsten Zeile angezeigt wird. Standardmäßig wird Text, der die Spaltenbreite überschreitet, abgeschnitten.

Element (Format) ViewDefinitions-Element (Format) anzeigen (Format)-Element TableControl-Element (Format) TableRowEntries Konfigurationselement für TableControl (Format) TableRowEntry-Element für TableRowEntries für TableControl (Format)-Wrap-Element für TableRowEntry für TableControl (Format)

## <a name="syntax"></a>Syntax

```xml
<Wrap/>
```
