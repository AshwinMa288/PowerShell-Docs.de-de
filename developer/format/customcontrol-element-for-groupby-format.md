# <a name="customcontrol-element-for-groupby-format"></a>Element „CustomControl“ für GroupBy (Format)

Definiert das benutzerdefinierte Steuerelement, das die neue Gruppe anzeigt.

Element (Format) ViewDefinitions-Element (Format) anzeigen (Format)-Element GroupBy Konfigurationselement für Ansicht (Format) CustomControl-Element für GroupBy (Format)

## <a name="syntax"></a>Syntax

```xml
<CustomControl>
  <CustomEntries>...</CustomEntries>
<CustomControl>
```

## <a name="attributes-and-elements"></a>Attribute und Elemente

Den folgenden Abschnitten werden die Attribute, untergeordnete Elemente und übergeordnete Element des Elements der `CustomControl` Element. Sie können eine beliebige Anzahl von untergeordneten Elementen angeben und auflisten in beliebiger Reihenfolge.

### <a name="attributes"></a>Attributes

Keine.
