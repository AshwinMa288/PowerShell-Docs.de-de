# <a name="enumeratecollection-element-for-expressionbinding-for-controls-for-configuration-format"></a>Element „EnumerateCollection“ für ExpressionBinding für Controls für Configuration (Format)

Angegeben, dass die Elemente der Auflistungen, die vom Steuerelement angezeigt werden. Dieses Element wird verwendet, wenn ein allgemeines Steuerelement zu definieren, das von allen Ansichten im die Formatierungsdatei verwendet werden kann.
