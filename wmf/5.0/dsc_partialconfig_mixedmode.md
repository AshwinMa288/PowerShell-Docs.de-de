# <a name="support-for-mixed-refreshmode"></a>Unterstützung gemischter „RefreshMode“-Eigenschaften

Wenn Sie Teilkonfigurationen verwenden, können nun mit einem `RefreshMode` einzelne Teilkonfigurationen definieren.
Weitere Informationen finden Sie unter [Teilkonfigurationen](https://msdn.microsoft.com/powershell/dsc/partialconfigs).
